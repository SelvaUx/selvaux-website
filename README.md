# selvaux.in — Personal Portfolio

> **Engineering Intelligence Beyond Limits**

My personal tech universe — Portfolio, Projects, Lab, Blog — all under one domain.

## 🚀 Live Site
🌐 [selvaux.in](https://selvaux.in)

## ✨ Features
- **Clean minimal design** inspired by exteraGram aesthetic
- **Custom cursor trail** — red dot following your mouse
- **Floating icon canvas** — shapes drifting in background
- **Typewriter effect** — cycling through roles
- **Scroll reveal animations** — fade-up on scroll
- **Counter animations** — stats count up with easing
- **Fully responsive** — mobile hamburger menu

## 📁 Structure
```
├── index.html          # Home — Hero, Projects, Skills, Principles
├── about/              # Bio, Skills, Strengths, Timeline, Vision
├── projects/           # SaraAI, SaraAI Max, OTDR-AI, Sara-OS, Voice File Manager
├── lab/                # Experiments (Coming Soon)
├── blog/               # Blog (Coming Soon)
├── assets/
│   ├── css/style.css   # Design system — CSS custom properties
│   └── js/script.js    # Cursor, canvas, typewriter, counters, scroll reveal

```

## 🎨 Design System
| Token | Value |
|-------|-------|
| Background | `#FFFFFF` (pure white) |
| Accent | `#F54142` (coral red) |
| Display Font | Syne |
| Body Font | Inter |
| Code Font | Space Mono |
| Border Radius | 28px (cards), 9999px (buttons) |

## 🛠 Tech Stack
- **HTML5** + **CSS3** + **Vanilla JavaScript**
- No frameworks. No build tools. No dependencies.
- Google Fonts via CDN
