## 👤 Overview

Welcome to my personal corner of the internet. This repository hosts my portfolio website, showcasing my journey in **Electronics and Communication Engineering (ECE)**, **VLSI**, **Embedded Systems**, and **AI Architecture**.

It serves as both a professional profile and a digital laboratory for my experiments.

**Live Site**: [selvaux.in](https://selvaux.in) 

---

## 🏗️ Project Structure

The website is architected as a multi-page application with a clear separation of concerns:

- **`/` (Home)**: A concise landing page featuring my "Empire Vision" and guiding principles.
- **`/about`**: Detailed biography, education, and skills matrix.
- **`/projects`**: Showcase of my technical projects and builds, including OTDR-AI.
- **`/lab`**: The experimental wing. A place for prototypes, demos, and mini-games.
  - **🤖 `/lab/saraai`**: **SaraAI** - An offline, intent-based AI behavior simulator.
  - **🎮 `/lab/cyber-snake`**: **Cyber Snake** - A neon-hacker style reimagining of the classic Snake game.
  - **🧬 `/lab/game-of-life`**: **Cellular Automaton** - Conway's Game of Life simulator.
  - **🌧️ `/lab/matrix-rain`**: **Cyber Rain** - Interactive falling code effect.
  - **🧩 `/lab/sandbox`**: **Sandbox** - Mini experimental projects.
- **`/blog`**: *(Coming Soon)* Thoughts on tech, engineering, and future realities.

## 🌟 Key Features

### 1. Modern, Dark-Themed UI
- Designed with a "hacker/developer" aesthetic in mind (`#0d1117` GitHub dark theme).
- Custom animations, including a 3D tilt effect on the profile picture.
- Fully responsive design using **Tailwind CSS**.

### 2. The Lab: Experiments & Mini-Games 🧪
A standout feature of this portfolio is the **Lab**, a place for prototypes and interactive demos:
- **🤖 SaraAI** (`/lab/saraai`): An offline AI behavior engine simulator demonstrating **intent-based system architecture**. Built with pure JavaScript, it features localized intent pattern matching and confidence scoring without external APIs.
- **🎮 Interactive Demos**: A collection of browser-based experiences including **Cyber Snake** (a neon-hacker style mini-game), **Cellular Automaton** (Conway's Game of Life simulator), and **Cyber Rain** (interactive matrix digital rain).

### 3. Technical Project Showcases 🚀
The **Projects** suite highlights complete, deployed projects:
- **OTDR-AI**: An AI-enhanced diagnostic system achieving 99.5% accuracy in classifying fault types in optical fibre networks using hybrid CNN-LSTM models.
- **Sara AI**: A Python-based intelligent virtual assistant to streamline daily tasks via web searches and system automation.

## 🚀 Running Locally

To view this project on your local machine:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/SelvaUx/selvaux-website.git
    cd selvaux-website
    ```

2.  **Start a local server**:
    *(Required because of ES6 modules used in SaraAI)*
    ```bash
    # Python 3
    python3 -m http.server 8000
    ```

3.  **Open in Browser**:
    Go to `http://localhost:8000`
